Design & Development
Keeping state in sync
React meets Web Audio API
The principal technical challenge of this small React project was to synchronize state between React and Tone.js, which is built on the Web Audio API. Since the Web Audio API knows nothing of React and maintains its own state in the browser, I had to use a number of React tricks to ensure that the states of the visual interface and audio engine matched at all times.
